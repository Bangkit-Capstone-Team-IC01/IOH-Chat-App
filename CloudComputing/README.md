# API List

## SignUp User
- URL
  - /user
- Method
  - POST
- Request body
  - name (string)
  - email (string)
  - password (string)
  - fpassword (string)
  - phone_number (string)
  - role (string)
- Response 

## Login User
- URL
  - /login
- Method
  - POST
- Request body
  - email (string)
  - password (string)
- Response
